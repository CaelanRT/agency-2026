# 22: Agentic Demo Flow and Polish

**Status:** todo
**Priority:** high
**Blocked by:** 20, 21
**Blocks:** none

## Description

Polish the app so the procurement dashboard and investigation copilot work together as one
coherent demo story.

## Acceptance Criteria

- [ ] Demo flow works:
  1. Open app
  2. See proactive watchlist of risky ministries
  3. Select or jump into a ministry
  4. Review recipients, risk indicators, and AI summary
  5. Ask an investigation question
  6. Read evidence-backed answer and follow-up suggestions
- [ ] The agent workflow feels grounded in live data, not generic chat
- [ ] Layout remains readable on desktop and acceptable on laptop-size screens
- [ ] Error states are handled for both scan and investigation flows
- [ ] The app tells a stronger "copilot" story than a report-viewer story

## Notes
